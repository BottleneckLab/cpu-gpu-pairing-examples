# CPU/GPU Pairing Examples

This repository contains educational CPU and GPU pairing examples for gamers and PC builders.

Created by BottleneckLab: https://bottlenecklab.com/

## Purpose

The goal is to help beginners understand why CPU/GPU balance changes by:

- resolution
- game type
- graphics settings
- refresh rate
- streaming workload
- CPU generation
- GPU class

## Important Note

These examples are not exact FPS predictions.

Real performance depends on many factors, including game engine, settings, RAM, VRAM, drivers, temperatures, and background apps.

## Example Pairing Factors

When reviewing a CPU/GPU combination, consider:

- Is the CPU strong enough for the target refresh rate?
- Is the GPU strong enough for the target resolution?
- Is the game CPU-heavy or GPU-heavy?
- Is the user gaming only or also streaming?
- Is the user targeting 1080p, 1440p, or 4K?
